PIX — Pagamento Direto

O cliente paga direto para sua conta bancária via PIX, sem intermediários. O QR Code e o código Copia e Cola são gerados automaticamente no checkout com a sua chave abaixo.
Cole exatamente como está cadastrada no seu banco
Aparece na tela do pagador (máx 25 caracteres)
Sem acento, maiúsculas (máx 15 caracteres)
Chave configurada: ✅ PIX ativo no checkout

OpenPix — Confirmação Automática de PIX

Como funciona: Quando configurado, cada cobrança PIX gera um QR Code único. Assim que o cliente paga, o OpenPix avisa o sistema automaticamente via webhook e o status do pedido muda para Pago sem intervenção manual.
📎 Crie sua conta grátis em app.openpix.com.br → Configurações → App IDs → Criar App ID 🔗 Webhook URL para configurar no OpenPix:
Nunca é enviado ao cliente. Fica seguro no servidor.
